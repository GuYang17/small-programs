# gyclock - A simple terminal clock program

## Introduction

gyclock is a lightweight terminal program based on ncurses tui lib.

## Usage

```bash
# Normal mode - shows time, date, and weekday
./gyclock

# Simple mode - time only
./gyclock --simple

# Show Tux with the clock
./gyclock --tux

# Simple mode with Tux
./gyclock --simple --tux

# Display help
./gyclock --help

# Show version
./gyclock --version
```
